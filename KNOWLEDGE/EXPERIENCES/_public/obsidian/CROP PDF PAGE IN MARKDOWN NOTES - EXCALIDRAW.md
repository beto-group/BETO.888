

## ENIGMAS
----

**Using Crop and Mask Image with PDFs in Markdown and Obsidian Canvas**:

- This feature works in markdown notes and on Obsidian Canvas.
- Ensure the PDF link includes the page reference in markdown.
- With the line active in markdown editor, select "Crop and Mask Image" from the Command Palette.
- Double-clicking the image will prompt you to open the crop file or the PDF page from where you've taken the image.
