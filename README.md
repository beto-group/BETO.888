## 🚀 Quick Start & Installation

To begin your journey with BETO.888, follow these simple steps:

1.  **Get Obsidian:** Ensure you have the [Obsidian.md](https://obsidian.md/) application installed on your device.
2.  **Download the Vault:** Download the latest `.zip` file for the BETO.888 Release from our official distribution point.
3.  **Unzip the File:** Extract the contents of the `.zip` file to a location on your computer where you want to store your vault.
4.  **Open in Obsidian:**
    *   Launch Obsidian.
    *   Click on **"Open another vault"** (the vault icon in the left-hand ribbon).
    *   Select **"Open folder as vault"**.
    *   Navigate to the folder you unzipped in the previous step and select it.

That's it! The BETO.888 vault will open, and your journey begins.

> [!TIP]
> The first note you'll see is this README. We recommend exploring the `[[Datacore.showcase]]` and the Bookmarked notes in the left panel to get oriented.
